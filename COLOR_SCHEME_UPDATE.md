### 🎯 設計特色

#### 1. **漸層運用**
- **進度條**: 寶石藍 → 琥珀金 → 珊瑚橘 (象徵光線流動)
- **主按鈕**: 寶石藍 → 學院深綠 (專業感)
- **CTA 按鈕**: 珊瑚橘 → 琥珀金 (溫暖邀請)
- **琥珀金按鈕**: 琥珀金漸層 (強調感)

#### 2. **透明度與陰影運用**
```css
/* 卡片背景 - 半透明白色搭配背景模糊 */
background: rgba(255, 255, 255, 0.95);
backdrop-filter: blur(20px);

/* 柔和擴散陰影 */
box-shadow: 0 8px 32px rgba(42, 82, 190, 0.15);

/* 互動反饋陰影 */
box-shadow: 0 15px 35px rgba(255, 191, 0, 0.50);
```

#### 3. **無障礙考量**
- ✓ 所有文字色彩對比度達 WCAG AA 標準
- ✓ 深色背景上使用亮白文字 (#FFFFFF)
- ✓ 琥珀金按鈕上使用深色文字 (#333333)
- ✓ 避免純黑白，使用暖色灰階

#### 4. **互動回饋**
- **Hover 狀態**: 10% 飽和度提升 + 陰影擴大
- **Active 狀態**: 輕微向下移動 (2px) + 陰影收縮
- **Disabled 狀態**: 50% 不透明度
