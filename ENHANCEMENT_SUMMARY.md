## 🚀 使用指南

### 瀏覽首頁（未登入）
1. 查看 Hero Section - 大標題 + CTA
2. 檢視倒數計時 - 距離校慶剩餘時間
3. 瀏覽活動亮點 - 3 個功能卡片
4. 查看活動時程表 - Timeline 組件

### 篩選攤位
1. 點擊地圖 → Map 頁面
2. 使用篩選按鈕：全部/美食/遊戲/手作
3. 點擊攤位卡片 → 彈窗詳情
4. 或直接點擊地圖上的藍框

### 時程更新
編輯 `Timeline.tsx` 中的 `TIMELINE_EVENTS` 陣列即可自訂活動時程。
