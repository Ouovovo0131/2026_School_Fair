# 🎪 校慶拾光地圖

這是一個為校慶活動設計的互動式網站。使用者可以從首頁查看活動亮點與倒數時間，切換到任務模式完成 20 關挑戰，也可以切換到地圖模式瀏覽整個校慶攤位配置與各區攤位資訊。

## 網站功能

- 首頁提供活動主視覺、倒數計時、活動亮點與時程表，方便快速掌握校慶重點。
- 任務模式共有 20 關，會依解鎖狀態顯示可進入的關卡、已完成關卡與未解鎖關卡。
- 完成任務後會累積進度，並可依完成數量解鎖小獎與大獎兌換資格。
- 地圖模式提供校園攤位總覽，可以依「全部、美食、遊戲、手作、其他」分類篩選攤位。
- 每個攤位卡片都能查看名稱、位置、簡介與亮點，搭配校園平面圖快速找到攤位位置。
- 登入後可同步個人進度，並顯示暱稱、頭像與完成比例。
- 管理功能可用來清除活動資料，適合現場活動管理使用。

## 使用方式

1. 進入首頁後，先查看倒數計時、活動亮點與活動時程。
2. 點選「前往玩遊戲」進入任務列表，查看各關卡的解鎖與完成狀態。
3. 點選已解鎖的任務卡片即可前往對應掃描頁面，依題目完成挑戰。
4. 完成任務後回到主畫面即可看到進度更新，累積到 10 關與 20 關時可進行兌換。
5. 若想先認識現場攤位，可切換到地圖模式，直接瀏覽校園攤位分布與分類。
6. 登入後可保存個人進度；未登入時也可以先瀏覽地圖與任務內容。

## 主要畫面

- 首頁：活動主視覺、倒數計時、亮點卡片、活動時程
- 任務頁：20 關任務清單、進度條、獎品兌換區
- 地圖頁：校慶攤位平面圖、分類篩選、攤位詳情
- 掃描頁：各關卡對應的任務內容與完成流程

## 玩法重點

- 先看首頁，快速了解活動時間與內容。
- 再進入任務模式完成關卡，累積進度與獎勵資格。
- 也可以先逛地圖，依攤位類型規劃動線。
- 完成指定關卡數後，依現場規則進行獎品兌換。

## 專案內容

- `app/components/Home.tsx`：首頁與主要互動入口
- `app/components/Map.tsx`：校慶攤位地圖與篩選介面
- `app/components/Countdown.tsx`：活動倒數計時
- `app/components/Timeline.tsx`：活動時程表
- `constants/quests.ts`：任務資料與題目設定
- `app/scan/[id]/page.tsx`：掃描後的任務頁面
