## 📊 色彩方案對比

### Before (舊方案)
- 背景: 純白/淺藍 (#ffffff / #f0f4ff)
- 主色: 蔚藍 (#0288d1)
- 輔色: 珊瑚紅 (#ff6b6b)
- 文字: 深藍/藍灰

### After (新方案) ✨
- 背景: 米白色/極淺暖灰 (#FDFBF7 / #F5F5F5)
- 主色: 寶石藍 (#2A52BE)
- 次色: 琥珀金 (#FFBF00) / 珊瑚橘 (#FF7F50)
- 文字: 深炭灰 (#333333) / 石板灰 (#4A4A4A)
- **優化**: 溫暖專業、高易讀性、校園感與慶典感平衡

## 🔧 核心更新內容

### 1️⃣ **CSS 根變數系統**

```css
:root {
  /* 米白背景 - 懷舊紙質感 */
  --background: #FDFBF7;
  
  /* 主色調 - 寶石藍系 */
  --primary-red: #2A52BE;      /* 寶石藍(正式) */
  --primary-yellow: #FFBF00;   /* 琥珀金(活力) */
  --primary-gold: #FF7F50;     /* 珊瑚橘(溫暖) */
  --primary-blue: #2A52BE;     /* 寶石藍(主要) */
  --primary-purple: #6B4C9A;   /* 深紫(搭配) */
  --primary-brown: #8B7355;    /* 暖褐(懷舊) */
  
  /* 文字色彩 - 高對比度 */
  --text-dark: #333333;        /* 深炭灰 */
  --text-secondary: #4A4A4A;   /* 石板灰 */
  --text-light: #757575;       /* 中灰 */
}
```

### 2️⃣ **背景漸層優化**

```css
/* 淺色模式 */
body {
  background: linear-gradient(135deg, #FDFBF7 0%, #F5F5F5 100%);
  /* 從米白→極淺暖灰，溫暖過渡 */
}

/* 深色模式 */
@media (prefers-color-scheme: dark) {
  body {
    background: linear-gradient(135deg, #1B1B1B 0%, #2D2D2D 100%);
    /* 深灰配色，保持對比度 */
  }
}
```

### 3️⃣ **文字色彩階層**

| 用途 | 色彩 | 範例 |
|------|------|------|
| **H1/H2 標題** | `#333333` | "2026 校慶拾光地圖" |
| **Body 內文** | `#4A4A4A` | 攤位說明、描述文字 |
| **輔助標籤** | `#757575` | 時間戳記、編號 |
| **淺色背景上** | `#FFFFFF` | 深色按鈕內的文字 |

### 4️⃣ **按鈕色彩系統**

#### 主按鈕 (寶石藍→深綠)
```css
background: linear-gradient(90deg, #2A52BE, #1B4332);
/* 現代專業感，象徵傳承 */
```

#### 琥珀金按鈕 (高亮促銷)
```css
background: linear-gradient(135deg, #FFBF00, #FF9500);
color: #333333;  /* 深色文字，確保易讀性 */
```

#### 珊瑚橘按鈕 (溫暖邀請)
```css
background: linear-gradient(135deg, #FF7F50, #FF6347);
/* CTA 呼籲行動用 */
```

### 5️⃣ **進度條漸層 (象徵光線流動)**

```css
.progress-bar-fill {
  background: linear-gradient(90deg, #2A52BE, #FFBF00, #FF7F50);
  /* 寶石藍→琥珀金→珊瑚橘
     象徵時間流動與「拾光」主題 */
}
```

### 6️⃣ **卡片與邊框設計**

```css
.premium-card {
  background: linear-gradient(135deg, #ffffff 95%, #FDFBF7 100%);
  border: 1.5px solid rgba(42, 82, 190, 0.15);  /* 寶石藍邊框 */
  box-shadow: 0 8px 32px rgba(42, 82, 190, 0.15);  /* 柔和陰影 */
}
```
